## Getting Started

Setup husky:

```bash
npm run prepare
```

Format code:

```bash
npm run format
```

Check invalid code:

```bash
npm run lint
```

## Development

.env file is required to run the project. For the full .env file example, check
out [this template](./template/.env.template) <br>
Then run the following command

```bash
npm run dev
```

Test app in production environment

```bash
npm run build
npm run preview
```
